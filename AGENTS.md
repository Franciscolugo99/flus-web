## Skills locales de Codex

- Usar `$impeccable` para auditoria visual, UX, accesibilidad, responsive y polish de pantallas existentes.
- Usar `$frontend-design` para crear o redisenar interfaces frontend completas.
- Antes de modificar UI, revisar los archivos reales del repo.
- No reemplazar pantallas enteras cuando el pedido sea solo corregir o mejorar una parte.
- No inventar arquitectura ni componentes inexistentes.
